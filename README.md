# PyFem
Python scripts for simple finite element analysis

# Requirement
- [Python3](https://www.python.org/downloads/)
- [Numpy](http://www.numpy.org/)
- [Scipy](https://www.scipy.org/)
- [Matplotlib](https://matplotlib.org/)

# Catagories
- [x] HeatConduct
- [x] Poisson 
- [ ] LinearElastic
- [ ] NonlinearElastic
- [ ] PhaseField
